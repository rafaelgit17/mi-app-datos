Incendio de ANDALUCIA

En la comunidad ANDALUCIA, provincia HUELVA, comarca CUENCA MINERA, municipio GRANADA DE RÍO-TINTO se quemó una superficie forestal de 143.6900. Hubieron 0 muertos y 0 heridos.
